Linux Automation Project

I developed an end-to-end automation script using Python in a Linux environment to process large Excel datasets and generate structured reports. The solution was executed within a virtual environment (venv), demonstrating practical experience in Linux-based automation, data processing, and scripting.

Problem Statement

Manual processing of multiple Excel sheets containing server data was time-consuming, repetitive, and prone to human error. The objective was to automate the extraction, filtering, and report generation process specifically for Linux-based systems.

Solution

Using Python (Pandas), I designed an automation workflow that:

  • Filters Linux servers from multiple sheets (N Line, N+1, Production)
  • Identifies and processes relevant system data dynamically
  • Generates structured Check list and Repo list reports
  • Splits reports based on patch windows for better operational handling
  • Exports multiple automated Excel files with organized sheets

Linux Usage

The automation was executed entirely in a Linux environment using virtual environments (venv). Linux command-line tools were used for environment setup, dependency management, and script execution, enabling efficient and scalable automation workflows.

Key Highlights

  • Automated multi-sheet Excel processing
  • Dynamic column detection using pattern matching
  • Scalable report generation for multiple environments
  • Reduced manual effort and improved accuracy

Execution in Linux

# Step 1: Create virtual environment
python3 -m venv myenv

# Step 2: Activate environment
source myenv/bin/activate

# Step 3: Install dependencies
pip install pandas openpyxl

# Step 4: Run automation script
